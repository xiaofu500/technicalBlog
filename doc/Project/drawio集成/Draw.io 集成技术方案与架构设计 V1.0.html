Draw.io 集成技术方案与架构设计 V1.0 ​

文档状态：草案
版本：V1.0
发布日期：2024 年 1 月
负责人：[技术负责人姓名]
适用团队：前端组、后端组、产品组
技术栈：Vue 3 + TypeScript + Vite + Pinia

1. 项目概述 ​

1.1 项目背景 ​

为满足产品项目中图表编辑、流程图绘制的需求，需要集成专业的绘图工具。经过评估，draw.io（现 diagrams.net）以其开源免费、功能强大、可定制性高等特点成为首选方案。

1.2 核心目标 ​

  • 在现有 Vue 应用中无缝集成 draw.io 编辑器
  • 支持图表的创建、编辑、保存、加载完整流程
  • 确保集成方案的安全性和性能
  • 为后续定制化功能预留扩展空间

1.3 非目标 ​

  • 不修改 draw.io 核心源码（除非必要）
  • 不实现实时协作功能（一期）
  • 不支持离线编辑（一期）

2. 技术选型与 Vue 集成分析 ​

2.1 集成方式对比 ​

方案优点缺点Vue 适配性
Iframe 嵌入隔离性好，安全性高，升级简单通信复杂，Vue 响应式数据传递需转换⭐⭐⭐⭐⭐
Vue 组件封装符合 Vue 开发习惯，响应式集成深度耦合，draw.io 升级影响大⭐⭐⭐
微前端方案完全独立，技术栈隔离架构复杂，通信成本高⭐⭐

2.2 最终决策 ​

采用 Iframe + Vue Composition API 方案，理由如下：

  1. Vue 友好：Composition API 更适合封装复杂通信逻辑
  2. 安全隔离：iframe 天然隔离，避免 draw.io 影响 Vue 应用
  3. 独立部署：可独立升级 draw.io 版本
  4. 性能优化：iframe 可懒加载，按需初始化

2.3 Vue 技术栈明细 ​

  • 核心框架：Vue 3.3+ (Composition API + <script setup>)
  • 状态管理：Pinia (模块化状态管理)
  • 构建工具：Vite 5.0+ (开发体验和构建性能优化)
  • 类型检查：TypeScript 5.0+
  • UI 框架：[Element Plus / Ant Design Vue / 项目现有 UI 库]
  • 通信方案：postMessage + CustomEvent + useDrawio Composables

3.2.1 DrawioEditor.vue (主编辑器组件) ​

vue
<template>
  <div class="drawio-editor-container">
    <!-- 自定义工具栏 -->
    <DrawioToolbar v-if="showToolbar" :disabled="!isReady" @save="handleSave" @export="handleExport" />

    <!-- 加载状态 -->
    <DrawioLoading v-if="loading" />

    <!-- Draw.io iframe容器 -->
    <div ref="iframeContainer" class="drawio-iframe-wrapper" :class="{ 'is-readonly': readonly }">
      <iframe
        v-if="shouldLoadIframe"
        ref="iframeEl"
        :src="iframeSrc"
        :title="title"
        allow="clipboard-read; clipboard-write"
        class="drawio-iframe"
        @load="handleIframeLoad"
      />
    </div>

    <!-- 错误提示 -->
    <DrawioError v-if="error" :error="error" @retry="initEditor" />
  </div>
</template>

<script setup lang="ts">
import { ref, computed, onMounted, onUnmounted, watch } from "vue";
import { useDrawioBridge } from "@/composables/useDrawioBridge";
import { useDrawioConfig } from "@/composables/useDrawioConfig";

const props = defineProps({
  diagramId: { type: [String, Number], required: true },
  readonly: { type: Boolean, default: false },
  autoSave: { type: Boolean, default: true },
  config: { type: Object, default: () => ({}) }
});

const emit = defineEmits<{
  save: [xml: string, svg?: string];
  loaded: [];
  "dirty-change": [isDirty: boolean];
}>();

// 状态管理
const iframeEl = ref<HTMLIFrameElement>();
const loading = ref(true);
const error = ref<Error | null>(null);
const isReady = ref(false);

// Composables
const { iframeSrc, buildConfig } = useDrawioConfig(props);
const { initBridge, sendMessage, loadDiagram, saveDiagram, destroyBridge } = useDrawioBridge(iframeEl);

// 计算属性
const shouldLoadIframe = computed(() => !props.readonly || props.diagramId);

// 生命周期
onMounted(async () => {
  await initEditor();
});

onUnmounted(() => {
  destroyBridge();
});

// 初始化编辑器
const initEditor = async () => {
  try {
    loading.value = true;
    error.value = null;

    // 初始化通信桥接
    await initBridge();

    // 配置draw.io
    const config = buildConfig({
      ...props.config,
      readonly: props.readonly
    });

    await sendMessage("configure", config);

    // 加载现有图表（如果有）
    if (props.diagramId) {
      const xml = await loadDiagramFromStorage(props.diagramId);
      await loadDiagram(xml);
    }

    isReady.value = true;
    emit("loaded");
  } catch (err) {
    error.value = err as Error;
    console.error("Draw.io初始化失败:", err);
  } finally {
    loading.value = false;
  }
};

// 保存处理
const handleSave = async () => {
  const result = await saveDiagram();
  if (result) {
    const { xml, svg } = result;
    emit("save", xml, svg);
    await saveToStorage(props.diagramId, xml, svg);
  }
};
</script>

3.2.2 useDrawioBridge.ts (通信层 Composable) ​

typescript
import { ref, onUnmounted, onMounted } from "vue";
import type { Ref } from "vue";

interface DrawioMessage {
  action: string;
  payload?: any;
  requestId?: string;
  timestamp: number;
}

interface DrawioResponse {
  success: boolean;
  data?: any;
  error?: string;
  requestId: string;
}

export function useDrawioBridge(iframeRef: Ref<HTMLIFrameElement | undefined>) {
  const messageHandlers = new Map<string, Function>();
  const pendingRequests = new Map<
    string,
    {
      resolve: (value: any) => void;
      reject: (reason?: any) => void;
      timeout: NodeJS.Timeout;
    }
  >();

  // 初始化通信桥接
  const initBridge = async () => {
    if (!iframeRef.value) {
      throw new Error("iframe未找到");
    }

    // 监听来自iframe的消息
    window.addEventListener("message", handleMessageFromIframe);

    // 等待draw.io加载完成
    await waitForDrawioReady();

    // 注册默认消息处理器
    registerDefaultHandlers();
  };

  // 发送消息到draw.io
  const sendMessage = <T = any>(action: string, payload?: any, timeout = 10000): Promise<T> => {
    return new Promise((resolve, reject) => {
      if (!iframeRef.value?.contentWindow) {
        reject(new Error("iframe未就绪"));
        return;
      }

      const requestId = generateRequestId();
      const message: DrawioMessage = {
        action,
        payload,
        requestId,
        timestamp: Date.now()
      };

      // 设置超时
      const timeoutId = setTimeout(() => {
        pendingRequests.delete(requestId);
        reject(new Error(`请求超时: ${action}`));
      }, timeout);

      // 保存请求上下文
      pendingRequests.set(requestId, {
        resolve,
        reject,
        timeout: timeoutId
      });

      // 发送消息
      iframeRef.value.contentWindow.postMessage(message, "*");
    });
  };

  // 处理来自iframe的消息
  const handleMessageFromIframe = (event: MessageEvent) => {
    // 安全验证
    if (event.source !== iframeRef.value?.contentWindow) return;
    if (!event.data || typeof event.data !== "object") return;

    const message = event.data as DrawioMessage | DrawioResponse;

    // 处理响应消息
    if ("requestId" in message && "success" in message) {
      handleResponse(message as DrawioResponse);
      return;
    }

    // 处理事件消息
    if ("action" in message) {
      handleEvent(message as DrawioMessage);
      return;
    }
  };

  // 加载图表
  const loadDiagram = (xml: string) => {
    return sendMessage("load", { xml });
  };

  // 保存图表
  const saveDiagram = () => {
    return sendMessage<{ xml: string; svg?: string }>("save");
  };

  // 导出图表
  const exportDiagram = (format: "png" | "svg" | "pdf" = "png") => {
    return sendMessage<{ data: string; format: string }>("export", { format });
  };

  // 清理资源
  const destroyBridge = () => {
    window.removeEventListener("message", handleMessageFromIframe);
    pendingRequests.forEach(({ timeout }) => clearTimeout(timeout));
    pendingRequests.clear();
    messageHandlers.clear();
  };

  return {
    initBridge,
    sendMessage,
    loadDiagram,
    saveDiagram,
    exportDiagram,
    destroyBridge
  };
}

3.3 Pinia 状态管理设计 ​

3.3.1 useDrawioStore.ts ​

typescript
import { defineStore } from "pinia";
import { ref, computed } from "vue";

export const useDrawioStore = defineStore("drawio", () => {
  // 状态
  const diagrams = ref<Record<string, Diagram>>({});
  const currentDiagramId = ref<string | null>(null);
  const isDirty = ref(false);
  const autoSaveInterval = ref<NodeJS.Timeout | null>(null);

  // getters
  const currentDiagram = computed(() => {
    return currentDiagramId.value ? diagrams.value[currentDiagramId.value] : null;
  });

  const hasUnsavedChanges = computed(() => {
    return currentDiagram.value?.hasUnsavedChanges || false;
  });

  // actions
  const setCurrentDiagram = (id: string, diagram: Diagram) => {
    diagrams.value[id] = diagram;
    currentDiagramId.value = id;
  };

  const updateDiagram = (id: string, updates: Partial<Diagram>) => {
    if (diagrams.value[id]) {
      diagrams.value[id] = { ...diagrams.value[id], ...updates };
    }
  };

  const startAutoSave = (diagramId: string, interval = 30000) => {
    stopAutoSave();

    autoSaveInterval.value = setInterval(async () => {
      if (hasUnsavedChanges.value) {
        await saveDiagram(diagramId);
      }
    }, interval);
  };

  const stopAutoSave = () => {
    if (autoSaveInterval.value) {
      clearInterval(autoSaveInterval.value);
      autoSaveInterval.value = null;
    }
  };

  return {
    // 状态
    diagrams,
    currentDiagramId,
    isDirty,

    // getters
    currentDiagram,
    hasUnsavedChanges,

    // actions
    setCurrentDiagram,
    updateDiagram,
    startAutoSave,
    stopAutoSave
  };
});

4. 通信协议设计 ​

4.1 消息类型定义 ​

typescript
// 请求消息类型
export type DrawioRequestAction =
  | "load" // 加载图表
  | "save" // 保存图表
  | "export" // 导出图表
  | "configure" // 配置编辑器
  | "undo" // 撤销
  | "redo" // 重做
  | "getState" // 获取状态
  | "customAction"; // 自定义操作

// 事件消息类型
export type DrawioEventType =
  | "init" // 初始化完成
  | "dirty" // 内容修改
  | "autosave" // 自动保存
  | "exportComplete" // 导出完成
  | "error"; // 错误发生

// 消息接口
export interface DrawioRequest {
  action: DrawioRequestAction;
  payload?: any;
  requestId: string;
  timestamp: number;
}

export interface DrawioResponse {
  success: boolean;
  data?: any;
  error?: string;
  requestId: string;
  timestamp: number;
}

export interface DrawioEvent {
  type: DrawioEventType;
  data?: any;
  timestamp: number;
}

5. 安全策略 ​

5.1 通信安全 ​

typescript
// 消息验证
function validateMessage(event: MessageEvent): boolean {
  // 1. 验证来源
  const allowedOrigins = [
    "https://embed.diagrams.net",
    window.location.origin // 自部署版本
  ];

  if (!allowedOrigins.includes(event.origin)) {
    return false;
  }

  // 2. 验证消息结构
  const requiredFields = ["action", "timestamp"];
  const data = event.data;

  if (!data || typeof data !== "object") {
    return false;
  }

  // 3. 验证时间戳（防止重放攻击）
  const timeDiff = Date.now() - data.timestamp;
  if (Math.abs(timeDiff) > 60000) {
    // 1分钟内的消息
    return false;
  }

  return true;
}

6. 性能优化方案 ​

6.1 懒加载策略 ​

typescript
// 按需加载draw.io
const useDrawioLazyLoad = () => {
  const shouldLoad = ref(false);

  const loadDrawio = () => {
    shouldLoad.value = true;
  };

  return {
    shouldLoad,
    loadDrawio
  };
};

// 在组件中使用
const { shouldLoad, loadDrawio } = useDrawioLazyLoad();

// 点击按钮或进入视图时加载
onMounted(() => {
  const observer = new IntersectionObserver((entries) => {
    if (entries[0].isIntersecting) {
      loadDrawio();
      observer.disconnect();
    }
  });

  observer.observe(containerRef.value!);
});

8.1 技术风险 ​

风险概率影响应对措施
draw.io 版本不兼容中高锁定版本，建立版本升级流程
