passport-jwt (登录鉴权) ​

用于登录鉴权的 jwt 登录

依赖 ​

typescript
@nestjs/passport
passsport-jwt
passport-local
@nestjs/jwt

secret 密钥 ​

typescript
export const jwtConstants = {
  secret: 'secretKey',
}

策略 ​

本地策略 ​

逻辑:在登录时，将用户名和密码通过 passport-local 策略进行验证，验证成功后，将用户信息存储在 session 中，并返回给客户端。

typescript
@injectable()
export class LocalStrategy extends PassportStrategy(Strategy) { 
      constructor(private authService: AuthService) {
        super({
          usernameField: 'username',
          passwordField: 'password',
        });
      }
      async validate(username: string, password: string): Promise<any> {
        const user = await this.authService.validateUser(username, password);
        if (!user) {
          throw new UnauthorizedException();
        }
        return user;
      }
}

jwt 策略 ​

逻辑:在登录成功后，将用户信息存储在 session 中，并返回给客户端一个 jwt token，这个 token 将在每次请求时作为认证信息，通过 passport-jwt 策略进行验证，验证成功后，将用户信息存储在 session 中，并返回给客户端。

typescript
@injectable()
export class JwtStrategy extends PassportStrategy(Strategy) { 
      constructor(private authService: AuthService) {
            super({
                jwtFromRequest: ExtractJwt.fromHeader('token'),
                ignoreExpiration: false,
                secretOrKey: jwtConstants.secret,
            })
      }
      async validate(payload: any) {
        return { userId: payload.sub, username: payload.username };
      }

}

使用 ​

注册到模块中 : 1. 在 auth.module.ts 中导入 LocalStrategy 策略，并将其添加到 providers 数组中。 2. 在 auth.module.ts 中导入 JwtModule 模块，并将 jwtConstants.secret 密钥注册为常量，并设置过期时间为 60 秒。

Typescript
@Module({
      imports: [PassportModule, JwtModule.register({
        secret: jwtConstants.secret,
        signOptions: { expiresIn: '60s' },
      })],
      providers: [AuthService, LocalStrategy, JwtStrategy],
      exports: [LocalStrategy],
})

注解到控制器 ​

Typescript
@Constroller('auth')
export class AuthController { 
      constructor(private authService: AuthService) { }
@UseGuards(AuthGuard('local'))
@Post('login')

async login(@Req() req: Request, loginDto: LoginDto) { 

      return {
            data: this.authService.login(loginDto),
            message: true,
      }
}

@UseGuards(AuthGuard('jwt'))
@Post('protected')
async protected(@Body() protectedDto: ProtectedDto) {
      return this.authService.protected(protectedDto);
      }
}
typescript
@Injectable()
export class AuthService {
      constructor(private jwtService: JwtService) { }
      async login(loginDto: LoginDto) {
        const user = await this.validateUser(loginDto.username, loginDto.password);
        if (!user) {
          throw new UnauthorizedException();
        }
        const payload = { username: user.username, sub: user.userId };
        return {
          access_token: this.jwtService.sign(payload),
        };
      }
      async protected(protectedDto: ProtectedDto) {
        return protectedDto;
      }
 }

jwt密钥 ​

公钥，非对称加密使用 用于加密 私钥，非对称加密使用 用于解密 加密解密流程:

  1. 客户端使用公钥加密数据，发送到服务器。
  2. 服务器使用私钥解密数据。
  3. 服务器使用私钥加密数据，发送到客户端。
  4. 客户端使用公钥解密数据。

如何防止中间人(Tampering)攻击:

  1. 服务器端验证解密后的数据是否合法。
  2. 客户端使用公钥加密数据，服务器端使用私钥解密数据。
  3. 服务器端使用私钥加密数据，客户端使用公钥解密数据。

https协议: ssl 证书：用于加密 https 请求，防止中间人攻击。 ssl 生成流程:

  1. 客户端向服务器端请求 ssl 证书。

  2. 服务器端生成 ssl 证书，并将其发送给客户端。

  3. 客户端验证 ssl 证书的有效性。

  4. 客户端使用 ssl 证书加密数据，发送到服务器端。

  5. 服务器端使用私钥解密数据。 创建密钥对:

  6. 服务器端使用 openssl 生成私钥和公钥。

  7. 服务器端将公钥发送给客户端。

  8. 客户端使用公钥加密数据，发送到服务器端。

  9. 服务器端使用私钥解密数据。
