面试问题 ​

1.渲染管线 ​

  • 流程 顶点数据->顶点着色器->图元装配->几何着色器（可选）->光栅化->片段着色器->逐片段操作->帧缓冲区
  • 各阶段操作详细说明
  1. 顶点着色器：对每个顶点执行的程序，用于计算顶点的位置、颜色、纹理坐标等属性。
  2. 图元装配：将顶点数据组织成图元（如点、线、三角形等）。
  3. 几何着色器（可选）：对图元执行的程序，用于生成新的图元或修改图元的属性。
  4. 光栅化：将图元转换为像素的过程，计算每个像素的位置、深度、颜色等属性。
  5. 片段着色器：对每个像素执行的程序，用于计算像素的最终颜色。
  6. 逐片段操作：对每个像素执行的程序，用于修改像素的颜色、深度等属性。
  7. 帧缓冲区：用于存储渲染结果的内存区域，最终会被显示在屏幕上。
